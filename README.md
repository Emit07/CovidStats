# covid

This is a a Vue App that will give you statistics about covid from the [COVID-19 api](https://github.com/javieraviles/covidAPI). This project was made so I could learn about view.

## What was Used?

* `Vue.js` (JS)
* `Bulma` (CSS)
* `axios` (API)

## Project setup
```
npm install
```
